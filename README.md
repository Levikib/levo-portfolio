## MotionGraphic вҖ” Technical Detail

The `/about` right panel runs a real-time canvas + DOM animation at 60fps. Architecture:

### Scramble Decode System

```
SCRIPT array  вҶ’  word groups with timing, colour, effects config
RAF loop      вҶ’  elapsed = performance.now() - seqStart
Per word      вҶ’  charsResolved = floor(elapsed / charMs)
Resolved      вҶ’  real character with word colour applied
Unresolved    вҶ’  random glyph from GLYPHS set (Latin, Greek, Japanese, Unicode blocks)
Word complete вҶ’  resolveFlash: brightness(3) spike fires
Chromatic     вҶ’  CSS animation: red/cyan text-shadow oscillates async
Shimmer       вҶ’  linear-gradient overlay sweeps at 250% width every 3s
Cursor        вҶ’  positioned after last active word, glows purple
Loop          вҶ’  700ms pause, then restarts from top
```

### Particle + Stream System

```
35 particles  вҶ’  random position, velocity, size, opacity
hexPulse      вҶ’  per-particle size breathing via sine wave (phase offset per particle)
7 streams     вҶ’  vertical data columns cycling through STREAM_COLORS
               вҶ’  ["#7c3aed", "#a855f7", "#10b981", "#4ead6a"]
Each frame    вҶ’  stream chars fall, fade, and cycle glyphs
```

### 3D Tilt

Mouse position normalised 0вҖ“1, applied to container per frame:

```js
tiltX += (targetX - tiltX) * 0.055   // lerped approach
tiltY += (targetY - tiltY) * 0.055
container.style.transform = `perspective(820px) rotateX(${tiltX}deg) rotateY(${tiltY}deg)`
```

Direct DOM mutation вҖ” zero React state, zero re-renders.
